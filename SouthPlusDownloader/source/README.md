# SouthPlus Downloader (南+论坛全自动资源下载解析器)

本程序是一款基于 C# 开发的自动化辅助工具，专为 South-Plus 论坛设计。通过它，您可以实现：绕过 Cloudflare 人机验证全自动获取账号 Cookie 凭证、无人值守批量解析多页帖子、自动扣除对应的购买费用、以及精准提取隐藏在代码片段内的网盘资源直链与解压密码等一站式解决方案。

## 核心特性

- 🤖 **全自动过盾抓取**：无需繁琐的抓包，内置微软原生的 Edge (WebView2) 浏览器。点击一键获取，程序会自动在后台开启浏览器访问论坛，当您通过 Cloudflare 人机或登录完成，瞬间一键抽离所有安全 Cookie 与匹配的真实 User Agent。
- ⚙️ **双模式支持**：支持传统友好的桌面图形界面，以及高效静默的终端命令行 (CLI) 运行环境。
- 💰 **智能消费风控**：您可以设置资源的购买价格上限（如最高 5 币），当检测到无资源被吞，或定价超过预算时，会自动跳过放弃，守住您的血汗钱。
- 🔍 **变态级正则提取引擎**：对于部分将密码只用四个数字隐藏（甚至不写防抽关键词），或直接将直链跟在前缀后（如 `https://pan.baidu.com...` 没有空格等）的疑难杂症帖，本工具的容错引擎均能完美捕捉提取。

## 运行与使用模式

### 1. 图形化界面运行 (GUI 模式)

如果在文件管理器双击运行 `SouthPlusDownloader.exe`（或者在终端不带任何参数启动），则会直接唤醒友好的图形化操作界面。

**使用流程：**
1. 启动程序后，在底部的【身份验证】框点击 **"自动获取 (内嵌浏览器)"**。
2. 耐心等待自带的微浏览器弹出，如果遇到「5秒盾」或者需要登录，请正常操作。
3. 一旦探测到您的登录态生效，弹窗将自动在 1.5 秒内销毁，并将截获的高价值 Cookie 等信息自动输入到界面里。**（此信息被自动保存在本地，下次双击打开可直接读取，无需重复获取）。**
4. 在上方选择板块、采集条数，设置你的购买上限和想要过滤的盘符（如 百度、夸克）。
5. 点击**开始采集**，左倚靠背，享受程序在日志框里为您不断输出硕果的快感！

---

### 2. 命令行自动挂机模式 (CLI 模式)

当您需要在服务器上挂机，或是结合其他的脚本工作流，可以使用命令行参数启动该程序。
> **前置条件：** 请务必至少成功运行一次 GUI 模式中的“自动获取”功能，以确保程序在根目录下成功生成合规的 `credentials.json` 令牌文件！命令行将直接依赖此文件通过身份验证。

```powershell
.\SouthPlusDownloader.exe [参数...]
```

**通用参数列表：**
*   `-b, --board`      目标板块的数字 ID。默认值 `216`。(对应 C104)
*   `-s, --start`      起始遍历条目 (如帖子列表的第 N 贴)。默认值 `1`。
*   `-e, --end`        结束遍历条目。默认值 `10`。
*   `-p, --price`      最高可接受自动付款的 SP 币价格上限。默认值 `5`。
*   `-k, --keywords`   必须包含的网盘名字，多选项使用英文或中文逗号隔开。默认抓取百度盘资源。
*   `-l, --log`        静默生成本地日志记录。在输出数据的同时写出 `southplus.log` 文件。
*   `-h, --help`       输出说明文件。

**使用示例**：
> 购买所有前三页内包含 百度、Mega、4004 的资源，并且每条最高限价 3 币，并写入文本日志：
```powershell
.\SouthPlusDownloader.exe -b 216 -s 1 -e 105 -p 3 -k "百度,mega,4004" -l
```

## 数据结果汇总

无论是图形界面还是命令行模式，最终买到的隐藏网链和提取码密码，以及原始原贴链接，全部都会被保存在同目录下的 `southplus_results.txt` 文件里。您可以再编写其他软件去定向消费其中的链接进行自动批量下载。
