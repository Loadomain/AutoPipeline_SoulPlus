# Baidu Netdisk Auto Downloader (百度网盘全自动解析下载仪)

这是一款带有图形界面（GUI）与命令行（CLI）双模式的百度网盘自动化批处理下载工具。借助内部强大的多轨“通道集束器”架构，支持完全无人值守、自动解决防盗链、自动填写提取码并抵抗百度网盘针对第三方开发者的流量风控。

## 🌟 核心特性

- **多通道防限流**：内置多条全球顶级开源网盘项目的公用 `AppKey`（如 AList 等），并在发生限流时自动热切换（Rotate），避免单通道因调用量过大报错。
- **独享私有通道**：若您自己申请了百度官方开发者权限，可强行走您个人的开发者私有专线，与外界流量彻底隔离。
- **WebView2 免密刮削**：底层采用微软新型 WebView2 沉浸式引擎，模拟人类真实浏览器自动过风控，解决各种复杂的分享链接提取码阻挡。
- **防打断熔断保护机制**：面对新机器、新账号可能弹出的机器验证码或服务条款弹窗，提供 60 秒的宽限期及自动界面浮出机制，方便您手工紧急处理。
- **支持断点续传（HTTP Range）**：下载超大体积文件突然中断或超时后，支持原地重试无缝拼接，突破服务器的强制截断限制。

## 🖥️ 1. GUI 桌面图形模式 (默认)

**如何启动：**
直接双击运行主程序 `BaiduAutoDownloader.exe` 即可进入可视化主界面。

### 1-1. 授权登录 (AuthForm)
首次使用时，必须点击上方**“1. 登录百度网盘授权”**。在弹出窗内，您可以自由选择使用：
*   **内置开源通道池（推荐）**：点击“应用”，即可用手机百度网盘 App 扫码登录。系统会帮您自动使用缓存的高速额度通道。
*   **私有开发者通道**：勾选后，在后方 `AppKey:` 留白处粘贴您的专属 Client ID（无须 Secret Key），点击应用即可登录绑定。

### 1-2. 构造任务并解析下载
准备一块符合 JSON 格式定义的任务列表文本（如下），粘贴进主界面左侧的大文本框中，并点击“2. 解析并开始下载任务”。
```json
{
  "我正在下载的文件A": {
    "地址": "https://pan.baidu.com/s/xxxx?pwd=abcd",
    "提取码": "abcd"
  }
}
```
引擎自动分离分享页数据、云端自动转存为您暂存区文件，最终转换为免限速 Dlink 直接存入你指定的硬盘路径下。

## 💻 2. CLI 命令行无头模式 (批处理与自动化集成)

为了方便与其他爬虫/脚本（如 Python 自动化爬虫管道、各类定时任务服务器等）无缝串联，下载仪被赋予了完全独立的命令行挂载模式。该模式将跳过所有的图形界面和人工干预，全程只将运行情况打印进终端和日志。

### 调用方法：
利用命令提示符 / Powershell，或者从外部程序中调用子进程：
```bash
# 基本调用，将接管解析 test.json，并直接把文件下载到 C:\tmp\test_download
BaiduAutoDownloader.exe --test D:\您的目录\config.json
```

### 注意事项（极度重要）：
*   **配置文件与 Token**：CLI 纯命令行本身**不支持弹出新窗口让您扫码登录**！因此，在您通过外部程序调用 CLI 前，强烈建议您**先由用户双击通过鼠标在图形界面完成至少 1 次登录**。获取到的持久化通行证会写入本地的 `config.json` 文件中，随后您再挂载调用 CLI，CLI 将静默读取文件并直接开跑。
*   **触发限流报错**：如果在无人值守的深夜，公用通道当日余额恰巧被耗尽。CLI 输出日志探测到 API 发回的 `errno: 4` 或 `-6`，会**立即熔断抛出 FATAL 退出程序**！请定期检查该程序的 `ExitCode` 和 `C:\tmp\cli_log.txt`。提示死机时，第二天手动打开一次图形界面切换新线路扫一下码即可恢复生产！
